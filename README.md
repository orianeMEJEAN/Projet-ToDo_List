# Project ToDo-List

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

It is recommended to read this README before cloning the 
project.

## Available Scripts

In the project directory, you can run:

### `npm start`

Open your folder and open a terminal in the root project folder, 
or open a terminal and navigate to the folder using the `cd` 
command.

The command will launch the application in development mode 
and open [http://localhost:3000](http://localhost:3000) in 
your browser.

### `npm install`

Please enter this command in the IDE terminal to retrieve 
the libraries used in the project.

### `npm install recharts`

Please also enter this second command in the 
IDE terminal to retrieve the chart library used in the 
project header.
